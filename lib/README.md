## Installation

```bash
npm install filtersprovider
```

### Peer Dependencies

```bash
npm install react@^19 react-dom@^19 react-router-dom@^7 @mui/material@^7 @emotion/react @emotion/styled
```

### Optional Dependencies

If your filters use virtualized lists:

```bash
npm install react-window
```

If your filters use MUI icons:

```bash
npm install @mui/icons-material
```

## Compatibility

- ✅ React 18 & 19
- ✅ React Router 6 & 7
- ✅ MUI 6 & 7
- ✅ Node 18+

## Test the build:

```bash
npm install
npm run build
```

## Publish:

```bash
npm login
npm publish --access public
```
